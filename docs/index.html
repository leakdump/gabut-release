Gabut

Version v1.0.10

Features

  • Certificate Transparency Monitoring: Monitor CT logs in real-time for newly issued SSL certificates
  • Domain Filtering: Filter domains by keywords to focus on relevant targets
  • Port Scanning: Discover open ports and services on targets
  • Service Fingerprinting: Identify web servers, technologies, WAFs, and CDNs
  • Vulnerability Scanning: Check for common security issues and misconfigurations
  • GeoIP Integration: Get geographical information about IP addresses
  • Notifications: Receive alerts via Telegram, Discord, Slack, email, or custom webhooks

Basic Usage

# Monitor Certificate Transparency logs
gabut monitor --keywords api,dev,staging

# Scan specific targets
gabut scan --target example.com --vuln

# Scan targets from a file
gabut scan --file targets.txt --output results.json
